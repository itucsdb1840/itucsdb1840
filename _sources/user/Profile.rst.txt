Profile
==========

**The profile page is only available after the user logs in, it becomes visible in the navigation bar when you log in**




      .. figure:: /figures/navbar.png
**The user can acces the profile page using My Profile button**
      .. figure:: /figures/profile.png
**In their profile the user can:**

* See their username
* See their current profile picture, it displays the default one if the user has not uploaded one yet
* Upload a new profile picture
