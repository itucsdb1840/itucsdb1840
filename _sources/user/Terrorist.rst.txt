Terrorist Groups
==========

**Any visitor, without logging in, has the ability to:**

Examine the terrorist group data where each terrorist group is displayed with it's respective:

* Name
* Number of total attacks it has made
* Number of total fatalities caused
* Number of total injuries caused
* Most Common Attack Target
* Most Common Attack Type
